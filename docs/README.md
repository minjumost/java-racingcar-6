# 기능 목록

- [x] 참여할 자동차 목록을 입력받는 기능
- [x] 전진을 시도할 횟수를 입력받는 기능

#### 다음은 한번의 전진 과정에 필요한 기능이다.

- [x] 1~9 사이의 숫자를 랜덤으로 생성하는 기능
- [x] 생성된 숫자를 분석하는 기능
    - [x] 4 이상이면 전진한다.
    - [x] 4 미만이면 전진하지 않는다.
- [x] 자동차 전진을 기록하는 기능

#### 한번의 전진 과정을 입력받은 횟수만큼 반복한다.

- [x] 기록을 바탕으로 순위를 매기는 기능
- [x] 순위를 출력하고 종료하는 기능
